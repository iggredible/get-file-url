# get-file-url

Use this to get file url from a file path! 

# Demo

```
getFileUrl('image.jpg'); // file:///some/path/to/image.jpg
getFileUrl('/some/path/to/image.jpg'); // file:///Users/path/to/image/get-file-url/demo/image.jpg
```
